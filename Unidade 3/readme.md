#**Deep Learning com PyTorch: Otimização Aplicada ao Fashion-MNIST**

##Este repositório contém os códigos e experimentos para a nota técnica aprofundada do Capítulo 6 do livro Deep Learning with PyTorch Step-by-Step, aplicando os conceitos de otimização de redes neurais ao conjunto de dados Fashion-MNIST.

###O objetivo é explorar e visualizar o funcionamento dos Otimizadores (como SGD e Adam) e Schedulers de Taxa de Aprendizagem, utilizando o Fashion-MNIST.
